☀️ SUNLYTICS (2024–2026)
It is a data analytics project that transforms Real - World solar panel sales data into actionable insights using Excel, Power BI, and AI tools. It focuses on data cleaning, visualization, and trend analysis to support business decisions like inventory planning, marketing strategy, and performance optimization.

>> Objectives

* Clean and preprocess real-time solar sales data.
* Analyze sales trends and performance patterns.
* Compare Monocrystalline vs Polycrystalline panels.
* Understand residential vs commercial segments.
* Build interactive dashboards for insights.

>> Tools & Technologies

* Microsoft Excel.
* Power BI.
* Microsoft Copilot (AI).
* Data Cleaning Techniques.
* Pivot Tables & DAX.

>> Key Features

* Real-time dataset analysis.
* Data cleaning and preprocessing.
* Interactive Power BI dashboard.
* Sales trend analysis (2024–2026).
* Product performance comparison.
* Market segmentation insights.
* AI-assisted analysis and reporting.

>> Key Insights

* Seasonal peaks in solar panel sales.
* Monocrystalline panels show higher efficiency.
* Residential segment contributes major revenue.
* Top-performing products identified for strategy.

>> Business Impact

* Improves inventory planning.
* Supports targeted marketing strategies.
* Helps in product performance evaluation.
* Enables data-driven decision-making.

>> Limitations

* Dashboard does not support real-time data refresh (static analysis).
* Predictive analytics (forecasting) is not implemented.
* Limited integration with external APIs or automated pipelines.

>> Future Scope
* Real-time data integration using APIs.
* Predictive analytics and sales forecasting.
* Advanced Power BI features (drill-through, KPIs).
* Web or app-based dashboard deployment.
